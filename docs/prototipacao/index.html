IFPB PPGTI

Projeto desenvolvido na disciplina de Engenharia de Sotware - Mestrado IFPB. Grupo 4-Hemocentro.

Informações

Plataforma que permite a notificação dos doadores do Hemocentro da Paraíba, buscando facilitar a demanda de comunicação do Hemocentro para aviso e chamamento para doação de sangue com doadores cadastrados. A notificação se dará via WhatsApp, de acordo com atributos como: o tempo mínimo de intervalo de doação de cada doador, a necessidade ou urgência de tipos sanguíneos específicos, notificando em tempo hábil, contemplando as necessidades de serviços e informações para doadores voluntários. 

Features

Principais funcionalidades

Doadores

Importação e cadastro de doadores do hemocentro para notificação de lembrete para doação pelo whatsapp.

Usuário

Gerenciamento de usuários para acesso ao sistema.

Notificação

Envio de notificações automatizada, por tipo sanguíneo e bairro, além do gerenciamento de notificações.

Histórico

Salvamento de histórico de operações, permitindo o monitoramento e reenvio de notificações.
